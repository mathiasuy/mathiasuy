Backend de tv.mb.uy

Backend para el sitio tv.mb.uy

Utiliza javscript con algunos plugins por ejemplo para la grilla. Se utilizó también php orientado a objetos, mysql, bootstrap 4, la api de YouTube. que verifica que los videos en vivo de los canales de Youtube que se dieron de alta en la base de datos estén online, y si lo están, lo marca como fuera de línea y busca otro de ese mismo canal, si lo encuentra, sustituye el video anterior que fue marcado como offline. También trae información relevante, como el logo del canal, nombre, descripción y cantidad de espectadores mirando. El sitio también maneja su propio contador de visitas por canal (mediante ajax). Todo se persiste en la base de datos mysql y se actualiza cada cierto tiempo mediante un corn (en el caso del rating por canal, cada algunos minutos y para el resto de la información cada cierto tiempo más largo) Sitio: ​http://tv.mb.uy​ URL del servicio: http://mtvuy.000webhostapp.com​. Código fuente: https://github.com/mathiasuy/crud-abm-seniales

  • Date: July 2017
